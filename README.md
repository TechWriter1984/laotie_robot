# laotie_robot
The Laotie(old fan) robot is a Raspberry Pi 4B based robot with caterpillar chassis, Pi camera, Pi Microphone, Google Coral USB accelerator.

## Features

- voice activated camera
- human face recognition
- caterpillar chassis
- chatbot
